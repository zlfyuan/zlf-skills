# 网络搜索技能 (web-search)

使用 curl 命令执行网络搜索和网页内容获取。

## 使用方法

```
/搜索 <关键词> [搜索引擎]
```

**搜索引擎优先级**：谷歌(默认) → 必应 → 百度

示例：
```
/搜索 今日新闻
/搜索 python教程 google
/搜索 微博热搜 bing
/搜索 天气预报 baidu
```

## 功能特点

- 支持三大搜索引擎：Google、Bing、Baidu
- 自动尝试 Google → Bing → Baidu 顺序（Google 优先）
- 自动处理反爬虫机制，使用合适的 User-Agent
- 支持获取任意 URL 的页面内容
- 智能解析搜索结果，提取标题和摘要

## 适用场景

- 实时新闻查询
- 百科知识搜索
- 网页内容抓取
- 热点话题追踪
- 技术文档查找
